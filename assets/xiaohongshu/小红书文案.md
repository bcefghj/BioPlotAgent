## 文案二：火山图教学帖（配图：02_volcano_detail.png + 01_volcano_plot.png）

### 标题
手把手教你看火山图｜零基础也能学会的生信绘图

### 正文
做完差异表达分析，火山图不会看？今天一次讲清楚！

🌋 什么是火山图？
想象站在火山口往下看——中间平坦，两侧隆起。
每个点=一个基因，越往两侧=变化越大，越往上=越显著

🔴 红色点 = 上调基因（实验组比对照组高）
🔵 蓝色点 = 下调基因（实验组比对照组低）
⚪ 灰色点 = 没有显著变化

虚线是阈值：
横线 = P值阈值（一般0.05）
竖线 = FC阈值（一般|log2FC|>1）

📊 这张示例图的结果：
- 5000个基因中
- 980个显著上调
- 918个显著下调
- 右上角标注了最显著的基因

💡 小贴士：
如果大部分点在中间灰色区域 → 两组差异不大
如果两侧彩色点很多 → 差异显著！

想一键生成这种图？用我开发的 BioPlotAgent！
上传你的DESeq2结果文件，3秒出图～

## 文案三：生存分析教学帖（配图：03_survival_detail.png + 04_survival_curve.png）

### 标题
TCGA生存分析不会做？5分钟学会看KM曲线

### 正文
做TCGA分析，生存曲线是必须要画的图！

📈 KM曲线（Kaplan-Meier）是什么？
追踪不同组别患者随时间的存活情况

怎么看：
- Y轴 = 生存概率（从100%往下掉）
- X轴 = 时间（月）
- 每下一个台阶 = 有患者去世
- 曲线下降慢 = 这组预后好
- 曲线下降快 = 这组预后差

📊 这张图的解读：
🔵 蓝线（低表达组）下降慢 → 存活更久
🔴 红线（高表达组）下降快 → 预后差
p = 0.0014 < 0.05 → 差异显著！
说明这个基因高表达与不良预后相关

下面的表格是"风险人数"（Number at Risk）：
各时间点还有多少人在观察中

用 BioPlotAgent 一键生成：
上传包含 time/event/group 的数据 → 自动画图+算P值

## 文案五：快速上手帖（配图：05_quickstart.png）

### 标题
3分钟安装！手把手教你用AI画生信图

### 正文
上篇介绍了 BioPlotAgent 这个工具，好多人问怎么安装？

来了来了！保姆级教程：

【准备工作】
确保电脑装了 Python（3.8以上）

【Step 1: 下载】
打开终端/命令行，输入：
git clone https://github.com/bcefghj/BioPlotAgent.git
cd BioPlotAgent

【Step 2: 安装依赖】
pip install -r requirements.txt
（等待几分钟，自动安装所有需要的包）

【Step 3: 启动】
streamlit run app.py
浏览器会自动弹出来！

【怎么用？】
左侧选"快速绘图" → 选图表类型 → 用示例数据或上传自己的 → 点击生成

就这么简单！
